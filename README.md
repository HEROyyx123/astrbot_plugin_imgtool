# astrbot_plugin_imgtool 🎨

**AstrBot 图片工具箱插件** — 集图片/GIF 旋转、对称翻转、变速播放、万花筒特效于一体的多功能工具。

## 功能一览

| 命令 | 功能 | 支持格式 |
|------|------|---------|
| `/旋转 <角度>` | 图片/GIF 任意角度旋转 | 图片 + GIF |
| `/对称 <方向>` | 水平/垂直/双向镜像翻转 | 图片 + GIF |
| `/变速 <倍率>` | GIF 播放速度调节 | 仅 GIF |
| `/万花筒 [段数]` | 经典三角对称万花筒特效 | 图片 + GIF |

## 效果展示

### 🌪️ 旋转 (`/旋转`)

任意角度旋转图片或 GIF，自动扩展画布显示完整内容。

```
/旋转 90      → 逆时针旋转90度
/旋转 -45     → 顺时针旋转45度
/旋转 180     → 旋转180度
```

### 🪞 对称 (`/对称`)

水平、垂直或双向镜像翻转。

```
/对称 水平     → 左右镜像翻转（默认）
/对称 垂直     → 上下镜像翻转
/对称 both     → 同时水平和垂直翻转
```

### ⏩ 变速 (`/变速`)

调整 GIF 播放速度，支持极高速丢帧策略和极低速帧插值，保证流畅度。

```
/变速 2.0     → 2倍速播放
/变速 0.5     → 半速播放（慢动作）
/变速 4       → 4倍速
```

### 🌸 万花筒 (`/万花筒`)

经典三角对称万花筒效果。可指定分割数和旋转动画增量，GIF 支持每帧旋转生成动态万花筒。

```
/万花筒           → 8段万花筒（默认）
/万花筒 12        → 12段万花筒，图案更细密
/万花筒 8 3       → 8段万花筒，每帧旋转3°，生成旋转动画
```

## 安装

### 方式一：WebUI 插件市场（推荐）

在 AstrBot WebUI → 插件市场 → 搜索 `imgtool` 安装。

### 方式二：手动安装

```bash
# 克隆到 AstrBot 的 data/plugins 目录
cd data/plugins
git clone https://github.com/yourname/astrbot_plugin_imgtool.git

# 重启 AstrBot 或在 WebUI 插件管理中加载
```

### 方式三：直接复制

将 `astrbot_plugin_imgtool` 文件夹复制到 AstrBot 的 `data/plugins/` 目录下，重启 AstrBot 即可。

## 使用方式

在聊天中发送图片/GIF（或回复一条图片/GIF消息），附带指令：

### 支持的触发方式

| 方式 | 示例 |
|------|------|
| **发送图片 + 指令** | 发一张图片，再发 `/旋转 90` |
| **回复图片 + 指令** | 回复一条图片消息，输入 `/对称 水平` |
| **图片链接 + 指令** | 发送 `/万花筒 8 https://example.com/image.jpg` |
| **同一条消息** | 部分平台支持图片和文字同一条消息发送 |

> ⚠️ **注意**：`/变速` 命令仅支持 GIF 格式图片。

## 配置

在 AstrBot WebUI 插件管理页面，可调整以下参数：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `max_frames` | int | `100` | GIF最大处理帧数，超过则均匀采样减少负担 |
| `kaleidoscope_segments` | int | `8` | 万花筒默认分割数（4-24的偶数） |

## 技术原理

### 万花筒算法（三角对称法）

1. 以图像中心为原点，将图像分割为 `N` 个等角扇形
2. 提取第一个扇形区域的内容
3. 对首扇区做镜像对折，使边缘自然衔接
4. 将镜像化后的基础楔形旋转复制到所有 `N` 个扇形位置
5. 裁剪回原始尺寸，形成完美对称的万花筒图案

GIF 支持逐帧递增旋转角度（`rotation_delta`），生成旋转动画效果。

### 变速策略

- **常规变速**（0.2x ~ 5x）：直接调整每帧延时 `duration / speed`
- **极高速**（>5x）：丢帧策略，按比例丢弃中间帧保证流畅
- **极低速**（<0.2x）：帧插值策略，重复帧并降低单帧延时防止超时
