# astrbot_plugin_imgtool 🎨

**AstrBot 多功能图片/GIF 工具箱插件** — 支持图片和GIF的旋转、对称、变速、万花筒和裸眼3D效果。

## 功能一览

| 功能 | 指令 | 说明 |
|------|------|------|
| 🔄 **旋转** | `/旋转 [角度]` | 任意角度旋转图片/GIF，如 `/旋转 90` |
| 🪞 **对称** | `/对称 [方向]` | 水平或垂直镜像翻转，如 `/对称 水平` |
| ⏩ **变速** | `/变速 [因子]` | 调整GIF播放速度，如 `/变速 2.0`（倍速） |
| 🔮 **万花筒** | `/万花筒 [段数]` | 对称分段式万花筒效果，如 `/万花筒 8` |
| 🕶️ **裸眼3D** | `/裸眼3d` | 分层假象裸眼3D效果 |
| 📖 **帮助** | `/图tool` | 显示所有指令 |

## 使用方式

发送一张图片/GIF（或回复一条图片/GIF消息），附带指令：

```
/旋转 90
/对称 水平
/变速 2.0
/万花筒 8
/裸眼3d
/图tool
```

## 效果原理

### 裸眼3D —— 分层假象法
1. **画白线分割背景** → 模拟「玻璃窗格」，制造层次感基底
2. **帧差法提取前景** → 利用GIF帧间差异自动识别运动主体
3. **前景覆盖在白线上** → 主体视觉上「跳出」白线平面，产生立体纵深感

### 万花筒 —— 对称分段法
1. 从图像中心裁切出一个扇形楔形块
2. 镜像该楔形块形成对称图案
3. 围绕中心旋转复制多份，拼接成完整的万花筒图案

## 配置

在 AstrBot WebUI 插件管理页面，可调整以下参数：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `line_spacing` | int | `80` | 裸眼3D白线间距(像素) |
| `line_width` | int | `3` | 裸眼3D白线宽度(像素) |
| `line_alpha` | int | `200` | 线条透明度 0-255 |
| `line_direction` | str | `both` | 线条方向: both/horizontal/vertical |
| `mask_threshold` | int | `25` | 前景检测阈值(0-255) |
| `mask_blur` | int | `7` | 前景边缘模糊半径 |
| `foreground_blur` | int | `0` | 前景高斯模糊(模拟景深) |
| `max_frames` | int | `48` | 最大处理帧数 |
| `kaleidoscope_segments` | int | `8` | 万花筒分段数 |
| `kaleidoscope_zoom` | float | `1.0` | 万花筒缩放比例 |
