# astrbot_plugin_imgtool 🎨

**AstrBot 多功能图片/GIF 工具箱插件** — 支持图片和GIF的旋转、轴对称、翻转、变速、GIF往返和裸眼3D效果。

## 功能一览

| 功能 | 指令 | 说明 |
|------|------|------|
| 🔄 **旋转** | `/旋转 [角度]` | 任意角度旋转图片/GIF，如 `/旋转 90` |
| 🪞 **对称** | `/对称 [方向]` | 轴对称：取图像一半对称到另一半，如 `/对称 上` |
| 🪞 **翻转** | `/翻转 [方向]` | 整体镜像翻转：水平/垂直，如 `/翻转 水平` |
| ⏩ **变速** | `/变速 [因子]` | 调整GIF播放速度，如 `/变速 2.0`（倍速） |
| 🔄 **GIF往返** | `/往返` | GIF正序播放结束后倒序播放再拼接，形成往返循环 |
| 🕶️ **裸眼3D** | `/裸眼3d` | 分层假象裸眼3D效果 |
| 📖 **帮助** | `/图tool` 或 `/img_help` 或 `/图帮助` | 显示所有指令 |

## 使用方式

发送一张图片/GIF（或回复一条图片/GIF消息），附带指令：

```
/旋转 90
/对称 上
/翻转 水平
/变速 2.0
/往返
/裸眼3d
/图tool
```

### 对称方向详解

| 方向 | 说明 |
|------|------|
| `/对称 上` | 保留上半部，垂直镜像到下半部 |
| `/对称 下` | 保留下半部，垂直镜像到上半部 |
| `/对称 左` | 保留左半部，水平镜像到右半部 |
| `/对称 右` | 保留右半部，水平镜像到左半部 |

### 翻转方向详解

| 方向 | 说明 |
|------|------|
| `/翻转 水平` | 整张图片左右翻转（镜像） |
| `/翻转 垂直` | 整张图片上下翻转（倒立） |

## 效果原理

### 裸眼3D —— 分层假象法
1. **画白线分割背景** → 模拟「玻璃窗格」，制造层次感基底
2. **帧差法提取前景** → 利用GIF帧间差异自动识别运动主体
3. **前景覆盖在白线上** → 主体视觉上「跳出」白线平面，产生立体纵深感

### GIF往返 —— 正序倒序拼接法
1. 提取GIF全部帧序列（帧 0, 1, 2, …, N-1）
2. 将序列逆序（去掉首尾重复帧，形成帧 N-2, …, 1, 0）
3. 正序 + 倒序拼接，最终帧序列为 0,1,2,…,N-1,N-2,…,1,0
4. 配合GIF的循环播放，呈现「正向→倒放→正向→倒放…」的往返效果

## 配置

在 AstrBot WebUI 插件管理页面，可调整以下参数：

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `line_spacing` | int | `80` | 裸眼3D白线间距(像素) |
| `line_width` | int | `3` | 裸眼3D白线宽度(像素) |
| `line_alpha` | int | `200` | 线条透明度 0-255 |
| `line_direction` | str | `both` | 线条方向: both/horizontal/vertical |
| `mask_threshold` | int | `25` | 前景检测阈值(0-255) |
| `mask_blur` | int | `7` | 前景边缘模糊半径 |
| `foreground_blur` | int | `0` | 前景高斯模糊(模拟景深) |
| `max_frames` | int | `48` | 最大处理帧数 |
